Quick Start

Drop three CDN links and the two Vectra files into your HTML <head> and you're ready to go — no build tools required.

html
<!DOCTYPE html>
<html lang="en" data-theme="dark">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <title>My App</title>

  <!-- Google Fonts -->
  <link rel="preconnect" href="https://fonts.googleapis.com" />
  <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
  <link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;500;600;700
             &family=JetBrains+Mono:wght@400;500&display=swap" rel="stylesheet" />

  <!-- Bootstrap 5 CSS -->
  <link rel="stylesheet"
        href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css"
        crossorigin="anonymous" />

  <!-- Font Awesome 6 -->
  <link rel="stylesheet"
        href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.6.0/css/all.min.css"
        crossorigin="anonymous" />

  <!-- AOS (optional scroll animations) -->
  <link rel="stylesheet"
        href="https://cdnjs.cloudflare.com/ajax/libs/aos/2.3.4/aos.css"
        crossorigin="anonymous" />

  <!-- Vectra CSS -->
  <link rel="stylesheet" href="assets/css/vectra.css" />
</head>
<body>

  <!-- your content -->

  <!-- Bootstrap JS (bundle includes Popper) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/js/bootstrap.bundle.min.js"
          crossorigin="anonymous"></script>

  <!-- AOS (optional) -->
  <script src="https://cdnjs.cloudflare.com/ajax/libs/aos/2.3.4/aos.js"
          crossorigin="anonymous"></script>

  <!-- Vectra JS -->
  <script src="assets/js/vectra.js"></script>
</body>
</html>
Vectra JS initialises everything automatically on DOMContentLoaded. No manual calls needed.

Design Tokens

All visual values — colours, radius, shadow, transitions — live as CSS custom properties on :root. Override any token in your own stylesheet after the Vectra link.

css
/* Example: change accent colour */
:root {
  --v-accent:     #7c3aed;  /* violet */
  --v-accent-2:   #a78bfa;
  --v-accent-rgb: 124, 58, 237;
}

Dark & Light Theme

Set data-theme="dark" or data-theme="light" on the <html> tag. Vectra JS persists the preference to localStorage.

HTML attribute

html
<!-- Dark (default) -->
<html data-theme="dark">

<!-- Light -->
<html data-theme="light">

Toggle button

html
<button data-toggle="theme" aria-label="Toggle theme">
  <i class="fa-solid fa-sun"></i>
</button>

JS API

js
// Toggle between dark and light
Vectra.toggleTheme();

// Explicitly set a theme
document.documentElement.setAttribute('data-theme', 'light');
localStorage.setItem('vectra-theme', 'light');

JavaScript API

vectra.js exposes a global window.Vectra object. All methods are initialised automatically — you can also call them programmatically.

Vectra.toggleTheme()Switches between dark and light theme. Saves preference to localStorage.
Vectra.scrollToTop()Smoothly scrolls the page to the top.
Vectra.initScrollSpy()Activates IntersectionObserver scroll spy on [data-scrollspy] nav elements.
Vectra.initSideNav()Activates sidebar scroll spy on [data-sidenav] nav elements.
Vectra.initCounters()Starts animated number counters on elements with data-counter.
Vectra.initCopyButtons()Wires .v-copy-btn to copy the nearest <pre> text to clipboard.
Vectra.initSliders()Activates all .v-slider elements with touch, swipe, and dot nav.
Vectra.initFileUpload()Enables drag-and-drop on .v-drop-zone elements.
Vectra.initFAB()Toggles the Floating Action Button action list.
Vectra.initCookieBanner()Shows/hides the cookie consent banner, persisted to localStorage.
Vectra.initPasswordToggle()Wires data-pw-toggle buttons to reveal/hide the target password input.

Data Attributes

Vectra JS activates features via data attributes — no manual JavaScript calls needed.

AttributeElementBehaviour
data-toggle="theme"<button>Toggles dark/light theme on click
data-scrollspy<nav>Activates top-navbar IntersectionObserver scroll spy
data-sidenav<nav>Activates sidebar IntersectionObserver scroll spy
data-counter="N"any elementAnimates number from 0 to N on viewport entry
data-autoplay="ms".v-sliderAuto-advances slider every N milliseconds
data-pw-toggle="id"<button>Reveal/hide password input with matching id
data-cookie-accept<button>Accepts cookies and dismisses banner
data-cookie-decline<button>Declines cookies and dismisses banner

Accessibility

Vectra is built with accessibility as a first-class concern.

  • Focus rings — all interactive elements display a clearly visible focus ring using --v-focus.
  • ARIA labels — icon-only buttons and toggles include aria-label attributes.
  • Reduced motion — all transitions and animations are disabled when prefers-reduced-motion: reduce is active.
  • Colour contrast — all text/background combinations in both themes meet or exceed WCAG AA (4.5:1 for normal text, 3:1 for large).
  • Keyboard navigation — modals, dropdowns, accordions, and tabs are all keyboard-accessible via Bootstrap's built-in handlers.
  • Semantic HTML — headings are hierarchical, navigation uses <nav> with aria-label, landmarks are used correctly.
  • Role attributes — switches carry role="switch", modals trigger role="dialog".

Customisation

The fastest way to customise Vectra is to override CSS variables in your own stylesheet, after the Vectra link.

Change accent colour

css
/* my-overrides.css — loaded AFTER vectra.css */
:root {
  --v-accent:     #7c3aed;
  --v-accent-2:   #a78bfa;
  --v-accent-rgb: 124, 58, 237;
  --v-accent-soft: rgba(124, 58, 237, .1);
  --v-accent-grad: linear-gradient(135deg, #7c3aed, #a78bfa);
  --v-accent-glow: 0 0 0 3px rgba(124, 58, 237, .3);
}

Change font

css
:root {
  --v-font: 'Geist', 'DM Sans', system-ui, sans-serif;
  --v-mono: 'Geist Mono', Consolas, monospace;
}

Extend with new components

Create a new CSS file and use the existing design tokens. Your components will automatically theme-switch with the rest of Vectra.

css
.my-widget {
  background: var(--v-surface);
  border: 1px solid var(--v-border);
  border-radius: var(--v-r);
  padding: 1.25rem;
  color: var(--v-text);
  box-shadow: var(--v-shadow-sm);
  transition: box-shadow var(--v-t);
}

.my-widget:hover {
  box-shadow: var(--v-shadow);
}
